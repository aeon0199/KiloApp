# KiloApp Private Beta Operations

## Cohort
- Target 10-30 users with macOS arm64 machines.
- Include power users with active Kilo CLI usage.

## Feedback channel
- GitHub issues for structured bugs.
- Include diagnostics bundle and diagnostics ID for every report.

## Response SLA
- P0: respond < 24 hours, patch within 24 hours.
- P1: respond < 48 hours, patch in next weekly drop.
- P2: triage weekly.

## Check-ins
- Day 2: installation + first-run friction.
- Day 7: session reliability + terminal feedback.
- Day 14: GA readiness review.

## Exit criteria for GA planning
- No unresolved P0 issues.
- P1 trend decreasing across two beta drops.
- Top 5 recurring pain points addressed or explicitly deferred.
